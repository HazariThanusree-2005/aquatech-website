# AquaTech Services Website

A clean, responsive single-page website for AquaTech Services — water filters and geysers.

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

## Build for Production

```bash
npm run build
npm run preview
```

## Stack

- React 18 + TypeScript
- Vite 6
- Tailwind CSS v4
- Framer Motion (animations)
- shadcn/ui components (Button, Card, Badge, Toast, Tooltip)
- Wouter (routing)
